Implementation/edu steps:
1. Init and setup git repository (project main branch, add license, readme, gitignore);
2. Init npm repository (edit package.json meta fields);
3. Setup editorconfig;
4. Write swagger spec;
5. Add js linter into the project (ESLint: https://eslint.org/docs/user-guide/getting-started)
6. Add needed app dependencies (express, mongoose);
7. Create hello world express server;
8. Start mongoDB as docker container;
9. Add mongoDB connection from your app (using mongoose);
10. Create mongoose Schema/Model;
11. Create express POST route to save user data to mongoDB.
